Analysera ny feedback

Så här gör du

Analysen sker i Claude Code — du klistrar in feedbacken direkt i chatten.

Steg för steg

  1. 1

    Öppna Claude Code på din dator.

  2. 2

    Klistra in feedbacken och skriv varifrån den kom, till exempel:
    "Analysera detta feature request från Intercom: [klistra in text]"

  3. 3

    Claude analyserar feedbacken, kontrollerar om det redan finns ett liknande ärende, och sparar resultatet i databasen.

  4. 4

    Kom tillbaka hit och ladda om sidan — ärendet syns direkt i nyhetsvyn och ärendelistan.

Tips: Du kan klistra in hela mejltrådar, Intercom-konversationer eller anteckningar från möten. Claude identifierar automatiskt om det är ett feature request eller något annat.
